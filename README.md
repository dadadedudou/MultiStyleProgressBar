# 技术预研
1. 自定义ProgressBar：支持endColor、startColor、centerColor自定义，实现渐变进度条效果
2. LottieAnimationView 加载json动画，并实现无缝切换
3. 自定义Loading
4. 文本显示：最多显示两条，第三条出现时，第一条向上滚动滑出
5. TextView显示文本，自适应高度，超过一定高度支持文本滑动
